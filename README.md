ポートフォリオサイトのサンプル（Next.js + TypeScript + Material-UI + MicroCMS）。

## セットアップ

```
$ cp .env.sample .env

NEXT_PUBLIC_MICRO_CMS_SERVICE_ID=
NEXT_PUBLIC_MICRO_CMS_API_KEY=
NEXT_PUBLIC_MICRO_CMS_WRITE_API_KEY=
```

環境変数をセット。

```
$ yarn install 
$ yarn dev
```

http://localhost:3000 にアクセス。
